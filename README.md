# 📚 我家的AI错题本

基于 **Streamlit** 的智能错题管理系统，支持多孩子账号、拍照录题、AI 分析、艾宾浩斯复习、中英双语学习等功能。

## ✨ 功能

- 👨‍👩‍👧‍👦 **多孩子** — 独立账号，不同年级，各自管理
- 📸 **拍照录题** — 豆包 Vision 自动识别+判科目+分析错因
- 🧠 **艾宾浩斯复习** — 自动安排 1/3/7/14/30 天复习计划
- 🤖 **中英双语** — 每日双语内容 + 英语听力（gTTS）
- 📅 **学习打卡** — 月度日历视图
- 📖 **错题本** — PDF 导出，题目答案分页
- 📊 **家长总览** — 统计图表
- 📥 **PDF 导出** — 复习题/错题/学习内容，A4排版
